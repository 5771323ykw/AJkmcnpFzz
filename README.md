# 基于Python的人脸表情识别系统 python856

## 项目概述

本项目是一个基于Python、结合PyQt5和深度学习技术的人脸表情识别系统。系统具备图片识别、视频分析、实时摄像头检测和表情分类等多种功能。

## 技术栈

- **GUI框架**: PyQt5
- **图像处理**: OpenCV
- **深度学习**: TensorFlow
- **神经网络**: DenseNet121
- **人脸检测**: face-recognition
- **数据处理**: NumPy
- **图像显示**: Pillow

## 功能模块

- **主窗口模块**
- **图片识别模块**
- **视频分析模块**
- **实时摄像头模块**
- **表情分类模块**
- **情绪概率可视化模块**
- **表情图标显示模块**
- **置信度分析模块**

## 系统角色

系统主要由上述提到的功能模块构成，各模块协同工作，实现表情的精确识别。
